Jsonp解决同源策略(同源/跨域)
2021.03.31
如果两个网址拥有相同的协议，域名，和端口，那么这两个网址就属于同一个源，其中只要 有一个不相同，就是不同源
jsonp 是 json with padding 的缩写，它不属于 Ajax 请求，但它可以模拟 Ajax 请求，原理如下
（1）因为前端引用 script 标签 是支持跨域的，所以我们把需要请求的数据的服务器地址 比如 xxx 用 < script src='xxx' > 引入，并且，前端必须要定义某个包含参数的方法 eg: test(data),其实data为你等下需要获取的远程数据
（2）服务器端 xxx 响应数据必须是一个函数的调用 test('需要传递的数据')，真正要发送给客户端的数据需要作为函数调用的参数
（3）所以当 引入 < script > 时，test(data)方法被调用。
