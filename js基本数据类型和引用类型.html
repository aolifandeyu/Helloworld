JS基本数据类型 及 == 和 === 的区别
2021.03.15
一、JS 内置类型有七钟:{null,undefind,number,boolean,string,object,symbol}又分为两大类：基本数据类型和引用数据类型
基本数据类型为 : number，string，boolean，null，undefind，symbol
引用类型为 : Array，function，Data，正则表达式 等等
基本数据类型：基本类型值在内存中占据固定大小，直接存储在[栈内存]中的数据
引用数据类型：引用类型在栈中存储了指针，这个指针指向堆内存中的地址，真实的数据存放在堆内存里。
二、typeof 对于基本类型，除了 null 都可以显示正确的类型，（对于 null 来说，虽然它是基本类型，但是会显示 object） 详细教程
三、双等于号和三等于号的区别和用法 详细教程

双等于号:

(1)如果两个值类型相同，则进行值的比较
(2)如果两个值类型不相同，也有可能相等，如果是字符串和数值比较，把字符串转化为数值之后再作比较，null 和 undefind 相等 eg:
let c="222"
let d=222
console.log(c==d) //输出true
(3)如果是引用类型的比较，变量保存的是数据的内存地址（指针），所以不同变量如果指向不一致，是不相等的 eg :
let a=[1,2,3,4];
let b=[1,2,3,4];
console.log(a==b) //输出false

let a=[1,2,3,4];
b=a
console.log(a==b) //输出true

三等于号:

(1)如果类型不同，就一定不相等 eg:
let c="222"
let d=222
console.log(c==d) //输出true,进行了转化
console.log(c===d) //输出false
(2)如果类型相同，则比较值是否相等
