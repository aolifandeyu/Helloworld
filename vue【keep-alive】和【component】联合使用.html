vue [keep-alive] 和 [component]的使用
2021.03.19
（1）、< component >元素，动态绑定多个组件到它的is属性上面:
< component is="index1">< /component>
< component :is="who">< /component>
（2）、< keep-alive > 保留状态，避免重新渲染：
< keep-alive>
< component :is="who">< /component>
< /keep-alive>
